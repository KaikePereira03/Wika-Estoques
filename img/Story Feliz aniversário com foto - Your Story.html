O editor do Canva contém as seguintes seções: navegação principal, painel lateral, barra de ferramentas, tela e controles de página. Para saber mais sobre cada uma dessas seções, pressione a tecla Tab. Para voltar à tela, aperte "Ctrl+F2".

A barra de ferramentas contém botões para editar o elemento selecionado. Para ir de um botão a outro, use as teclas de direita e esquerda. Para ativar o botão, aperte Enter. Para voltar à tela, aperte "Ctrl+F2".
A tela, ou editor, é onde fica seu design. Para acessar a página, aperte Enter. Para acessar os grupos ou elementos da página, aperte Enter novamente. Clique na aba para acessar a próxima página, grupo ou elemento. Para usar a barra de ferramentas, use a tecla "Ctrl+F1". Se um elemento estiver selecionado, você pode movê-lo usando as teclas de seta; para girar, use as teclas "," e "."; para redimensionar, use as teclas de seta junto com a tecla "Ctrl". Você pode aumentar o incremento desses atalhos com a tecla "Shift".
